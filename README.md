# Multi-Agent-Paper-Reading-Assistant
随着大模型能力提升，单一 Agent 已经可以完成基础问答和总结任务，但在复杂任务场景下，单 Agent 往往存在以下问题：  任务过于复杂，单次推理链条长，容易遗漏信息 不同子任务需要不同“专业角色”处理，如元数据提取、方法总结、公式解释、参考文献梳理 输出结构不稳定，可解释性弱，难以工程化扩展 面对长篇论文时，容易出现上下文溢出、重点丢失、解释粒度不一致等问题  为解决这些问题，本项目设计一个基于 Supervisor 模式的多 Agent 协作系统。系统允许用户上传论文 PDF，由总控 Agent 负责任务分发与流程调度，多个专家 Agent 分工协作，最终产出结构化的论文精读报告。
